### 一、两数之和

##### ---代码(python3实现)

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        dict={}
        for i,n in enumerate(nums):
            a= target - n
            if a in dict:
                return [dict[a],i]
            dict[n]=i
```



### 二、有效的括号

##### ---代码(python实现)

###### 第二次提交

```python
class Solution(object):
    def isValid(self, s):
        """
        :type s: str
        :rtype: bool
        """
        d ={')':'(','}':'{',']':'['}
        stack = []
        for char in s :
            if char in '({[':
                stack.append(char)
            elif char in ')}]':
                if not stack :
                    return False
                else:
                    if stack.pop() != d[char]:
                        return False
        if stack :
            return False
        else :
            return True
```

### 三、转换成小写字母

##### ---代码(python3)

```python
class Solution:
    def toLowerCase(self, s: str) -> str:
        return s.lower()
```

##### ---代码(c语言)

```c
char * toLowerCase(char * s){
    int len = strlen(s);
    for (int i = 0; i < len; ++i) {
        s[i] = tolower(s[i]);
    }
    return s;
}
```
